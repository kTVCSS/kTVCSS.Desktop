# kTVCSS Desktop Application

Electron приложение для kTVCSS.

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

## Сборка установщика

```bash
npm run build
```

Установщик будет создан в папке `dist/`.

## Особенности

- Приложение не засыпает во время работы
- Поддержка системных уведомлений
- Автоматическая загрузка звуков
- Разрешение окна по умолчанию: 1720x900
- Установщик для всех пользователей системы
